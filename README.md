# calculadora-subredes
Herramienta web para el cálculo de subnetting en redes IPv4. Implementa lógica de direccionamiento IP, conversión binaria y segmentación por bits prestados, generando un desglose completo de subredes, incluyendo máscara, CIDR, hosts asignables, broadcast y rangos de direcciones. Desarrollada con JavaScript y estilizada con Tailwind CSS.
